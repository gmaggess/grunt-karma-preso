Grunt

- JavaScript Task Runner.


- Analogous to ANT.

- Powered by NodeJS. That is, it's just an npm module.


- Browser and platform agnostic, minify CSS, process SASS, optimize images, run tests...


- Who uses it?! Adobe, Twitter, Google, jQuery, and us!
